# 4.5 Goal with the Test

The objectives of the verification campaign for Project **A1** are:

* **Verify and validate** all test cases and product functions.
* **Focus on re-testing, verifying, and validating** failed test cases and functions carried over from previous iterations or prototypes:
  * Incorrect or missing functions
  * Interface errors
  * Behavior or performance errors
