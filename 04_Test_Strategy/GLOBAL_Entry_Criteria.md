# 4.7.1 Entry Criteria

Before starting any testing activities or phase, the following conditions must be met:

* **Test environment** availability and readiness.
* **Test tools** setup and validated in the test environment.
* **Testable code** availability (firmware build or software release package).
* **Test data** availability (inputs, configuration profiles, test sheets).
