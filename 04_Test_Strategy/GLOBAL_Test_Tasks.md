# 4.10 Test Tasks

The responsibilities and daily tasks for the test-responsible persons include:

* **Plan** all test activities and campaign timelines.
* **Execute** manual and automated test suites.
* **Report** faults, bugs, and anomalies to the appropriate tracking systems and project stakeholders.
* **Gather** and compile test execution logs and results into formal test reports.
* **Formulate** statements of compliance based upon the verified system requirements.
* **Participate** actively in project alignment meetings to prevent design failures before test execution.
* **Develop** new test cases, test specifications, and verification methodologies.
* **Maintain** existing test scripts and physical test fixtures.
* **Prune** and remove obsolete or redundant test cases.
