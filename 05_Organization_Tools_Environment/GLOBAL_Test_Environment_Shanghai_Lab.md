# 5.3.1.1 Test Facilities: Shanghai Lab

The Shanghai laboratory is equipped with the following test systems and facilities:

* **Air-Performance & Suction:** Air-performance box, pressure-meter, stopwatch, tachometer.
* **Filter & Dust Handling:** Clogging equipment, filtration-efficiency measurement setup.
* **Acoustics:** Dedicated noise-level room.
* **Cleaning Performance:** DPU (Dust Pick-Up) room.
* **Lifetime & Durability Rigs:**
  * Move-bed (10 workstations)
  * Reciprocating with impact on nozzle (2 workstations)
  * Fishing equipment rig (1 workstation)
  * Package drop test rig (1 workstation)
  * Vibration table (1 workstation)
  * Switch lifetime tester (1 workstation)
  * BRC (Brush Roller Control) lifetime rig (1 workstation)
  * Tumble tester for charger/IEC (1 unit)
* **Environmental Chambers:** Temperature & humidity chambers (2 units).
* **Electrical Instruments:**
  * Oscilloscopes
  * DC power sources
  * Multimeters / current meters
  * Agilent data loggers / sensors
