# 5.2 Test Management Tools

The standard corporate toolset for managing testing activities, test cases, and defects is:

* **Pythia:** Used for writing test specifications, storing execution logs, generating summary test reports, and tracking overall test case status.
* **Jira:** Used for defect management, sprint task tracking, and mapping issues back to the respective development pipelines.
