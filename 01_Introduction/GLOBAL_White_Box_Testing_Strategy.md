# 1.3.1 White Box Testing Strategy

White box testing is a testing method in which the internal structure, design, and implementation of the item being tested is known to the tester. These tests can be functional or non-functional.

> [!NOTE]
> White box testing allows for detailed path coverage, code inspection, and structural security analysis, ensuring that the internal logic is robust and optimized.
