# 2.3 System Overview – Nozzle

The Nozzle assembly is the primary motorized cleaning interface. It supports interchangeable roller configurations—including standard bristle, fluffy soft-roll, and wet clean models—enabling optimized pickup across hard floors, carpets, and wet surfaces.
